# MATLAB



## Desktop Shortcut
1. create file `MATLAB.desktop` with contents
```bash
[Desktop Entry]
Version=1.0
Type=Application
Name=MATLAB
Exec=/opt/MATLAB/R2020b/bin/matlab
Icon=/opt/MATLAB/R2020b/resources/coreui/matlab/splash.png
Terminal=false
```
2. save it to `/usr/share/applications`.
